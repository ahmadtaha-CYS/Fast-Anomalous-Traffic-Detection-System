# Fast Anomalous Traffic Detection System 🚗📡

## 📖 About The Project
This project focuses on the security of vehicular communications. It features a modeled vehicle control system used to analyze potential cyber threats. The core of the project is a designed system to detect network anomalies and prevent data injection attacks. 

## 🛠️ Tools & Technologies
* **Network Analysis:** Wireshark, Nmap, Tcpdump
* **Programming & Scripting:** Python, Bash
* **Core Concepts:** Cyber-Physical System Security, Anomaly Detection, Traffic Analysis

## 🚀 Project Goals
* Model vehicular network behaviors to establish a baseline for normal traffic and analyze potential cyber threats.
* Detect network anomalies and prevent data injection attacks.
* Evaluate the speed and efficiency of the anomalous traffic detection system.
